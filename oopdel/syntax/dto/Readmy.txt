Класс является каркасом (шаблоном) для создания обьектов
Обьект является программой (экземпляром класса)

По шаблону (класс) может быть создано 
сколько угодно экземпляров класса (обьектов)

Из чего состоит класс

1. Из полей
Примет класса, состоящего только из полей, приведен в Typefield.php

Такая структура называется Data Transfer Object (DTO) - Обьект передачи данных
Происходит использование обьекта вместо массива, чтобы передать много значений
Может использоваться для обмена данными между модулями программы
